# Claude Code Learning Logs

A non-technical founder's daily account of building an AI operations system from scratch.

**Written by:** [Danielle Vantini](https://www.linkedin.com/in/danielle-vantini-w-carboni-5007b556/)
**Started:** March 22, 2026
**Tools:** Claude Code, Claude AI, GitHub, Make.com
**Location:** San Diego, CA

## What this is

Every session I build something, break something, or learn something.
I write it down here — in plain English, no jargon.

This is not a highlight reel. It is a working log.

## Who this is for

Non-technical founders, marketing operators, and business owners
who want to understand what it actually looks like to build with AI —
not the polished version, the real one.

## The system being built

13 agents. A transcription pipeline. A daily intelligence report.
Automated GitHub backups. A course in progress.
All of it documented, entry by entry, in this repo.
