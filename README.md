# Claude Code Learning Logs

A non-technical founder's daily account of building a 23-agent AI operations system from scratch using Claude Code.

**Written by:** [Danielle Vantini](https://daniellevantini.com)
**Positioning:** One of the earliest non-technical business adopters of AI — documented proof dating to 2021.
**Started:** March 23, 2026
**Tools:** Claude Code, Claude (Anthropic), GitHub, Make.com
**Location:** San Diego, CA

## What this is

Every session I build something, break something, or learn something.
I write it down here — in plain English, no jargon.

This is not a highlight reel. It is a working log.

## Who this is for

Non-technical founders, marketing operators, and business owners who want to understand what it actually looks like to build with AI — not the polished version, the real one.

## The system being built

23 agents. A transcription pipeline. A daily intelligence report.
Automated GitHub backups. Hooks that enforce CLAUDE.md rules with teeth.
A course in progress: *From Zero to AI Operations Architect*.
All of it documented, entry by entry, in this repo.
